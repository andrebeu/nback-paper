# nbackEM
Model of the nback task using context based episodic memory mechanisms.

* utilsEM, utilsWM and utilsCommon contain the model source code, and some helper functions for running experiments and generating model figures. 
* trainWM is a notebook you can use to train a new instance of the WM network
* runEM-barplts is a notebook you can use to replicate the model results from the paper
